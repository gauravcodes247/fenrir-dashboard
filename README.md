A simple security dashboard built as part of a 24-hour technical screening task.
The project includes  signup form, dashboard analytics, dark mode support, and a scan management table(static).\


Tech Stack:
1.React (Vite) 
2.React Router v6 – Routing
3.Tailwind CSS – Styling & dark mode
4.MUI DataGrid – Table component
5.React Icons – Icons
6.Vercel – Deployment

Setup Instructions:
1.Clone the repository : git clone https://github.com/gauravcodes247/fenrir-dashboard
2.Install dependencies : npm i
3.Start development server: npm run dev


Deployment
The project is deployed on Vercel.

Known Limitations
1.Authentication is currently UI-only
2.Data in dashboard and tables is static/mock data
3.No backend integration yet
